# 基金实时估值工具（重仓股模拟）

通过基金的前十大重仓股及其占比，结合股票实时涨跌幅进行加权计算，模拟基金当日估值。

## 快速开始

### 一键启动（推荐）

双击 `start.bat` 文件即可：
- 自动安装依赖（首次运行）
- 启动代理服务器
- 打开浏览器

### 停止服务

双击 `stop.bat` 文件停止所有服务

### 手动启动

1. 安装依赖：
```bash
npm install
```

2. 启动代理服务器：
```bash
npm start
```

3. 打开浏览器访问 `index.html`

## 核心原理

1. 获取基金的前十大重仓股及持仓占比
2. 实时获取这些股票的涨跌幅
3. 按持仓占比加权计算：`估算涨跌幅 = Σ(股票涨跌幅 × 持仓占比)`

## 功能特点

- ✅ 基于真实重仓股数据计算
- ✅ 实时获取股票行情
- ✅ 显示详细持仓明细
- ✅ 自动每分钟刷新
- ✅ 数据本地保存
- ✅ 一键启动/停止

## 使用说明

1. 输入基金代码（如：110022）和持有金额
2. 点击"添加基金"
3. 系统自动获取重仓股并计算估值
4. 点击"查看持仓"可展开前十大重仓股明细

## 数据来源

- **基金持仓**：东方财富网基金档案
- **股票行情**：新浪财经实时行情接口

## 注意事项

- 估值仅基于前十大重仓股，实际基金还包含其他持仓
- 重仓股数据通常每季度更新一次，可能存在滞后
- 股票行情在交易时间（工作日 9:30-15:00）实时更新
- 最终收益以基金公司公布的净值为准

## 示例基金代码

- 110022 - 易方达消费行业
- 161725 - 招商中证白酒
- 320007 - 诺安成长混合
- 001102 - 前海开源国家比较优势
